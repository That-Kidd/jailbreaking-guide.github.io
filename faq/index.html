Jailbreaking FAQ



What is Jailbreaking?

Jailbreaking is an exploit used to elevate permissions and gain root filesystem access to iOS. This allows installation of custom unsigned code.

What iOS versions are supported?

Not all versions of iOS are supported and not all devices are supported on certain versions. To check which Jailbreak supports your device click here.

Is it possible to restore to a specific iOS version?

Yes it is possible provided you have saved blobs. Learn more about blobs here.

Is it possible to remove a Jailbreak?

Yes it is possible to remove a Jailbreak. For more info click here.

Does this tweak work with my device?

Not all tweaks work with every device on every iOS version. If you have download the recently released Chimera Jailbreak, there is a Chimera Tweak Compatibility List. If that doesn't help you can ask on the r/Jailbreak Subreddit.

Is Chimera compatible with Cydia or Substrate?

Chimera is not compatible with Cydia or Substrate and instead uses Sileo and Substitute. Some alternative jailbreaks are Electra for Sileo, Cydia, and Substitute; Unc0ver for Cydia and Substrate; Unc0ver Dark for Sileo, Cydia and Substrate.

Will Chimera ever support Cydia?

Coolstar has elected not to support Cydia anymore. However, some developers are working to get Cydia working alongside Sileo 1.0.

What's the difference between Tethered, Untethered, and Semi-Untethered

Tethered Jailbreaks require a constant connection to a computer. These jailbreaks are typically for developers only and are not recommended for most.

Untethered Jailbreaks, while a thing of the past, only require a connection to a computer once. These jailbreaks do not require a Kickstart (Re-Jailbreak) from a computer or app to keep functioning. The last Untethered Jailbreak was Pangu9 on iOS 9 - 9.0.2.

Semi-Untethered Jailbreaks are much more common these days and require a Kickstart upon rebooting your iOS device.
