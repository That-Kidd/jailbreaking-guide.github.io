Jailbreaking FAQ



What is Jailbreaking?

Jailbreaking is an exploit used to elevate permissions and gain root access to iOS. This allows installation of apps and tweaks.



What iOS versions are supported?

Not all versions of iOS are supported and not all devices are supported on certain versions. To check which Jailbreak supports your device click here.



Is it possible to remove a Jailbreak?

Yes it is possible to remove a Jailbreak, either by restoring to an iTunes backup or by updating to an unsupported version of iOS.



How can I remove my Jailbreak?

For Jailbroken devices on iOS 7.1 - 10.3.3 you can use Cydia Eraser If you are using BytaFont, it is best to uninstall before using this. For devices running iOS 11 using the Electra Jailbreak you can use Rollectra. Alternatively, you can restore to an iTunes backup, restore the Factory Settings using iTunes, or update ot an unsupported version of iOS.
