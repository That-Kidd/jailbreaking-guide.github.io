Pangu Jailbreak Method

This method only supports 64-Bit Devices on iOS 9.2 - iOS 9.3.3. If you are on a newer version of iOS 9, try updating to iOS 10.3.3 or iOS 11.3.3. If you are using a 32-Bit device please use Phoenix instead.
Before continuing this guide, please make sure you have backed up your iOS Device via iTunes and iCloud.
