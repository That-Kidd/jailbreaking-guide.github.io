Star Jailbreak Method

This method only supports iOS 1.0 - 4.0.1 devices, and may work better with Windows. Step 4's instructionsare made for Windows and may be different on macOS.
Before continuing this guide, please make sure you have backed up your iOS Device via iTunes and iCloud.
For this guide you will need the following:
  • Wi-Fi Access (WPA/WPA2 with password works too)
  • Charles Proxy
  • (OPTIONAL/UNTESTED) iFunBox
Step 1: Setup Proxy with Charles
  1. Open Charles Proxy (download link above)
  2. In the menu bar, choose Proxy > SSL Proxying Settings > Add
  3. Press Add and set both fields to * and press OK, then press OK again
If this fails, try connecting your device to the Internet via Ethernet instead of WiFi, or try a Windows PC.
Step 2: Connect to Proxy
  1. Open Settings > Wi-Fi > Press the Info button next to your Wi-Fi network
  2. Scroll to the bottom, and select Manual for HTTP Proxy
  3. Visit https://www.whatismyip.com/ on your computer and put your Local IP in for the Server field on your device, then put 8888 into the Port field.
  4. Open Safari, and visit "charlesproxy.com/getssl" and install the certificate.
If this doesn't work, try using your Public IPv4 adress.
Charles only runs for 30 minutes, and you must be on your local network to use it, so open Charles and repeat this step before using Cydia.
You may see a message on your computer asking if you would like to allow your device to connect. Press Allow.
Step 3: Install Star and Cydia
  1. Open Safari, and visit jailbreakme.com/star
  2. Slide the Slide to Jailbreak just like you are unlocking your device, and wait for it to finish.
  3. Reboot your device
You may want to set Auto-Lock to Never to stop the device from sleeping.
Step 4: Fix "untrusted server certificate" (UNTESTED)
  1. Open Charles again
  2. Repeat Step 2
  3. Open Cydia, then do a Complete Upgrade
  4. Install PreferenceLoader from Cydia
  5. Download the .deb from https://github.com/nabla-c0d3/ssl-kill-switch2/releases/tag/0.7
  6. Download the newest iFunBox at http://www.i-funbox.com/en_download.html
  7. Switch to the "My Device" tab of iFunbox and click Install Deb and drag-and-drop the above mentioned .deb.
  8. Reboot your device
  9. Go to Settings, then scroll down and see if SSL Kill Switch 2 appears. If it doesn't, try following this guide instead which is similar. (Tap on the word this.)

Please note that every time you must use Cydia, you need to launch Charles and repeat Step 2. When Charles is not open or is disabled (the trial only lasts 30 minutes, then you must restart the app to start it again), you have no Internet connection once you follow Step 2 so if you don't plan to use Cydia for a extended amount of time, you may want to switch back to no HTTP Proxy.
